# CLAUDE.md

## Project Overview

This is an LLM agents project for agent reinforcement learning research.

## Development

- Python project using `pyproject.toml` for configuration
- Main entry point: `main.py`

## Commands

Use `uv run` to run any python scripts

```bash
# Install dependencies
uv run python -m pip install -e .

# Run the main script
uv run python main.py
```
